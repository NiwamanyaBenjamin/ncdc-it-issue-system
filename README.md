# NCDC IT Issue Resolution Management System

Professional role-based IT Issue Resolution Management System for NCDC.

## Run locally

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Deployment

Deploy on Render and connect Neon PostgreSQL using the `DATABASE_URL` environment variable.

## Main requirements covered

- Users report IT issues
- Manager assigns issues to available IT Staff
- IT Staff updates progress and uploads resolution reports
- User confirms resolved issues
- Admin creates, approves, disables users and assigns roles
- Notifications, audit logs, reports, knowledge base, profile management and backup records
